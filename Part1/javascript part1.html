Q5. How many way we can create variables in javascript, give name convention for javascript variable.

We can create variables by using var keyword.
There are few general rules to create variable:
1. Variable names should contain letter, digits and underscore and dollar signs
2. Name should always start with letter
3. Reserved keywords cannot be used for variable names.

Q6. What is hoisting in javascript ? What is advantages and disadvantages for hoisting. How we can avoid that

Hoisting is default behavious of Javascript's of moving declarion of all the variable at top of the current scope before code execution.
we can use the variable before it is getting declared or without declaring it, we can assign value to the variable

Advantages:
script will not throw and error even if you have missed to declare any variable and assigned value to it.
disadvantages: all variables which are hoisted are getting created at the global scope, so those can be accessible globally.
