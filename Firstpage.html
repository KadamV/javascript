Q1. What is pseudo class and pseudo elements?

Pseudo-class: is used to define a specific state of an element. e.g. unvisited, visited, mouse hover, selected. A CSS pseudo-element: is used to style specified parts of an element. e.g style the fist letter or insert any content after or before of any element.

Q2. Give examples of all pseudo elements.
  1. first-line
  2. first-letter
  3. before
  4. after
Q3. What is use of before and after...
'before' and 'after' pseudo-elements are used to insert the content before or after the content of particular element respectively.
Q4. What is Opacity, give an examples...
Opacity specifies the transperency of an element. Opacity has a default initial value of 1
